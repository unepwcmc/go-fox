# Go-Fox

## Requirements
- Rails 5.1.4
- Ruby 2.5.0p0
- Postgres 9.6.3
- Reactjs
- Webpacker

## Setup

- `bundle install`
- `rails db:create db:migrate db:seed`
- `yarn install`
- `brew install redis`
- `brew install graphviz`

You'll need to open a separate tab for each of these to run in...

- `rails s`
- `webpack-dev-server`
- `redis-server`
- `sidekiq`
